# desafio-fullstack-veritas
Desafio Fullstack–Mini Kanban de Tarefas (React + Go) O objetivo deste desafio é avaliar a sua capacidade de desenvolver uma aplicação fullstack simples utilizando React no frontend e Go no backend. Você deverá implementar um pequeno Kanban com três colunas fixas e documentar o fluxo de uso do sistema em um diagrama.
